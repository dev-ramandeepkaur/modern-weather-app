🌦️ Modern Weather Dashboard
Get real-time weather updates with a clean and simple interface.

I built this project to practice Vanilla JavaScript and learn how to work with APIs. The goal was to create a weather tool that isn't just functional, but also looks great with a modern "Glass-style" design.

✨ Key Features
Live Search: Enter any city name to get instant weather updates.

Default Location: Automatically loads weather for Delhi, India on startup.

Modern UI: Uses "Glassmorphism" (blur effects) for a premium look.

Dynamic Icons: Weather icons change automatically based on the current sky conditions.

Extra Details: View Humidity, Wind Speed, and Pressure along with the Temperature.

🛠️ Built With
HTML5 & CSS3: For the structure and responsive design (using Flexbox and Grid).

JavaScript: For fetching data and updating the screen without refreshing.

OpenWeather API: The source for all real-time weather information.
